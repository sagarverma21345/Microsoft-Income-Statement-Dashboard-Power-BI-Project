# Microsoft-Income-Statement-Dashboard-Power-BI-Project
Built an interactive Power BI Dashboard to analyze Microsoft’s Income Statement data, including Revenue, Net Income, Operating Income, and EPS. Used DAX, data modeling, and visualization techniques to create KPI cards, trend charts, and financial reports for business insights and performance analysis.
This project is an interactive Power BI dashboard built using Microsoft’s financial data to analyze the company’s Income Statement performance across multiple fiscal years.

The dashboard provides insights into:

Revenue growth trends
Net Income and Operating Income analysis
Cost structure evaluation
Year-over-Year financial comparison

It is designed to help users quickly understand Microsoft’s financial health using clear visualizations and KPI cards.

🛠️ Tools & Technologies Used
Power BI
Microsoft Excel / CSV
Data Cleaning & Transformation
DAX Measures
Data Visualization
📈 Dashboard Features
Interactive Income Statement Table
KPI Cards for:
Net Income
Total Revenue
Operating Income
Revenue vs Net Income Comparison Chart
Multi-year Financial Trend Analysis
Clean and Professional UI Design
📊 Key Insights
Microsoft showed consistent growth in Total Revenue across fiscal years.
Net Income increased significantly, indicating improved profitability.
Operating Income trends highlight strong operational efficiency
